# Steam Turbine Predictive Maintenance

## Overview

This project investigates predictive maintenance techniques for steam turbine operational data using machine learning and deep learning approaches. The objective is to detect abnormal operating conditions and support condition-based maintenance.

## Methods

- Exploratory Data Analysis (EDA)
- Missing Value Analysis
- Outlier Detection
- Correlation Analysis
- KNN Regression
- Autoencoder-based Anomaly Detection
- PCA + Mahalanobis Distance
- SARIMA Forecasting

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- TensorFlow / Keras
- Statsmodels
- Matplotlib
- Seaborn

## Features Analyzed

Examples of monitored parameters include:

- Bearing Vibration
- Bearing Temperature
- Steam Pressure
- Steam Temperature
- Rotor Position
- Governor Valve Position
- Turbine Speed
